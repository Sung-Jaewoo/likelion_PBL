# 📘 Today I Learned

### 1. 오늘 배운 내용
-JS로 DOM을 조작해서 카드 데이터를 화면에 동적으로 렌더링
-입력 폼에서 값을 받아서 새로운 카드와 상세 정보를 추가하는 기능 구


### 2. 핵심 정리 (내 언어로)
-이번에 느낀 건 JS에서 DOM을 어떻게 다루냐가 진짜 핵심이라는 거다.
HTML은 그냥 틀일 뿐이고, 실제로 화면을 바꾸는 건 결국 JS다
getElementById, createElement, appendChild 이런 걸 쓰면서
→ 내가 직접 HTML을 만들어서 집어넣는 느낌을 처음으로 이해했다
특히 중요한 흐름은
=>데이터 -> JS -> DOM 생성 -> 화면 출력
이거 하나로 정리되는 것 같다.
그리고 입력창에서 값을 받아서 그걸 다시 카드로 만들어서 DOM에 넣는 과정이 반복되니까 “아 이게 웹이 돌아가는 구조구나”라는 걸 좀 감 잡았다.
