# 📘 Today I Learned

### 1. 오늘 배운 내용
-기존 3주차 코드 위에 기능을 추가하는 방식으로 확장
-랜덤 데이터 추가 (1명, 5명) 기능
-필터링 (파트별), 정렬 (최신순), 검색 기능
-fetch를 사용해서 외부 API 데이터를 불러오기
-로딩 / 성공 / 실패 상태를 UI로 표시하는 방법


### 2. 핵심 정리 (내 언어로)
-4주차 핵심은 데이터를 가공해서 보여주는 것이다.
그냥 보여주는 게 아니라,
1.필터링
2.정렬
3.검색
이런 과정을 거쳐서 최종 화면이 만들어진다.
그리고 fetch를 쓰면서 외부 데이터까지와의 연결을 체감할 수 있었다.
물론 상태 관리도 중요하다는 걸 느꼈다. (=>로딩 / 에러 처리 안 하면 UX가 망가짐)

### 4. 느낀 점
-3주차는 '단순 만들기 느낌'이었다면, 4주차는 '딥한 상용 서비스 운영 구축 느낌'이었다
특히 필터랑 검색 구현하면서 생각보다 로직이 복잡해져서 좀 헷갈렸는데, 그래도 하나씩 나눠서 하니까 결국 성공적으로 완성했다.
그리고, fetch를 쓰면서 이제 진짜 데이터 받아와서 서비스 구축 단계로 넘어가는 느낌이 들었다.
아직 코드 정리는 좀 더 해야할 것 같긴 한데 그래도 흐름을 몸소 확실히 체감한 것 같다.
